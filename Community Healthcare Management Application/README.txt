To run the file, datechooser.jar file is required to be imported for the date combobox functionality used in the project.

Once the file runs, the SYSADMIN username and password both are "sysadmin".
Created a System wherein the System Admin can create User Accounts for Community, Hospitals, Doctors and Patients. 
SystemAdmin can perform CRUD on Hospitals, Communities, Doctors and Patients.
CommunityAdmin can furhter create/update new Communities.
HospitalAdmin can further create/update doctor's and patient's information under the hospital.
Doctors can view their patient's information, accept appointments scheduled by patients and update notes/vital signs.
Patients can view/edit their personal information and send an appointment schedule request by searching for a doctor under a hospital under any community.
